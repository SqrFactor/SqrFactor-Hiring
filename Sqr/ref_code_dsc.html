Referal Code Description



How does the Referral Code works?


1) Complete the Registration Process (Example - X,Y and Z register as Team Members)
2) Complete the Transaction/Payment Process.
3) After you have completed the Payment, you will receive an email with the Team Registration code and Referral codes for each team member.
4) Example -
Team Member 1(X) will have referral code as Team Registration Code(Example : G16KA13S001)+ 01 = G16KA13S00101 will be the Referral code for Team Member 1.
Similarly for Team Member 2(Y) and Team Member 3(Z) will have their referral code as Team Registration Code(Example : G16KA13S001) +02 and +03 respectively.

So the Referral Codes for the Team Members will be :
Team Member 1 = Team Registration code(Example : G16KA13S001) + 01 = G16KA13S00101
Team Member 2 = Team Registration code(Example : G16KA13S001) + 02 = G16KA13S00102
Team Member 3 = Team Registration code(Example : G16KA13S001) + 03 = G16KA13S00103

Note* - For Individual Registration, there will be only 1 referral code. For Team of 2 there will be 2 referral codes and For Team of 3 there will be 3 referral codes.Above given codes are just examples

How does it benefit me?


The Team member gets a referral code which they can refer to their friends to get a recharge coupon worth Rs. 50(Only if their referral code is used by their friends/Team).

How does it benefit new teams/individuals registering for the competition?


The Team/Individual who uses the referral code while registering on our website will get an off of Rs. 300. And also, after they have successfully registered and successfully paid the amount, they will get a referral code which they can refer to their friends.
